# Travel Agency System - Basic Exception Handling

## What This Project Does
A simple travel booking system where users enter data through the keyboard.
Shows how to handle errors when users type wrong information.

### 1. try-catch-finally blocks
```java
try {
    // Code that might cause an error
} catch (Exception e) {
    // What to do if error happens
} finally {
    // Always runs (cleanup code)
}
